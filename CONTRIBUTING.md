# Como Contribuir

Obrigado pelo interesse em contribuir com "Anatomia de um Agente de IA".

## O que você pode fazer

### Revisão de conteúdo
- Corrigir erros técnicos nos capítulos do núcleo
- Melhorar clareza e precisão das explicações
- Identificar inconsistências entre capítulos

### Novas seções
- Camadas analíticas (governança, regulação, VSM) — ver [ROADMAP](ROADMAP.md)
- Camadas práticas (workflows, receitas, casos de uso)
- Tradução para espanhol

### Ferramentas
- Melhorias na Cartografia C0 (`cartografia/c0-visao-geral.jsx`)
- Scripts de extração de dados (`dados/`)

## Como contribuir

1. Faça um fork do repositório
2. Crie uma branch: `git checkout -b minha-contribuicao`
3. Faça suas alterações
4. Abra um Pull Request com descrição clara do que foi mudado e por quê

## Padrões de escrita

- Português brasileiro (PT-BR)
- Cada capítulo termina com seção `## Implicações`
- Headers: H1 para título, H2 para seções principais, H3 para subseções
- Citações de código em blocos de código com linguagem especificada
